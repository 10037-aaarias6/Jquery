Adentrate sobre el concepto básico de Siblings

Método Siblings():

Podemos atravesar los lados en el árbol Dom con la ayuda de jQuery y encontrar los hermanos de un elemento.

Los elementos hermanos comparten el mismo padre.

Traversing Sideways

Los métodos jQuery para recorrer los laterales en el árbol dom son:

  • siblings()
  • next() & nextAll()
  • nextUntill()
  • prev() & prevAll()
  • prevUntil()

    • Siblings

    El siblings() es un método incorporado en jQuery que se utiliza para encontrar todos los elementos hermanos del elemento seleccionado.

    Su Sintaxis es:

    $(selector).siblings(function)

    Ejemplo:

    
            <!DOCTYPE html>
            <html>
              
            <head>
                <style>
                    .block {
                        display: block;
                        border: 1px solid black;
                        padding: 10px;
                        color: black;
                    }
                </style>
                <script src=
            "https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js">
              </script>
                <script>
                    $(document).ready(function() {
                        $("h3").siblings().css({
                            "color": "green",
                            "border": "3px solid green"
                        });
                    });
                </script>
            </head>
              
            <body>
                <div class="block">
                    <h2 class="block">Geeks</h2>
                    <h3 class="block">for</h3>
                    <h4 class="block">Geeks</h4>
                </div>
            </body>
              
            </html>

    Resultado


    Método Next():

    El next() es una función incorporada en jQuery que se utiliza para devolver el siguiente hermano del elemento seleccionado.

    Su Sintaxis es:

    $(selector).next()

    Ejemplo:

    
            <html>
  
            <head>
                <style>
                    .next * {
                        display: block;
                        border: 2px solid lightgrey;
                        color: black;
                        padding: 5px;
                        margin: 15px;
                    }
                </style>
                <script src=
            "https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js">
              </script>
                <script>
                    $(document).ready(function() {
                        $("h3").next().css({
                            "color": "black",
                            "border": "2px solid green"
                        });
                    });
                </script>
            </head>
              
            <body class="next">
                <div>
                    This is parent element !
                    <p>This is first paragraph
                  </p>
                    <span>first span box </span>
                    <h2>heading 2 !</h2>
                    <h3>heading 3 !</h3>
                    <p>This is the second paragraph
                      and next sibling to h3 !</p>
                </div>
            </body>
              
            </html>

    Resultado


    Método nextUntil():

    El nextUntil() es un método incorporado en jQuery que se utiliza para encontrar todos los elementos hermanos entre dos elementos dados.

    Su Sintaxis es:

    $(selector1).nextUntil(selector2)

    Ejemplo:

    
            <html>
  
            <head>
                <style>
                    .bet_sib * {
                        display: block;
                        border: 2px solid lightgrey;
                        color: black;
                        padding: 5px;
                        margin: 15px;
                    }
                </style>
                <script src="https://ajax.googleapis.com/ajax/libs/
                            jquery/3.3.1/jquery.min.js"></script>
                <script>
                    $(document).ready(function() {
                        $("span").nextUntil("p").css({
                            "color": "black",
                            "border": "2px solid green"
                        });
                    });
                </script>
            </head>
              
            <body class="bet_sib">
                <div>
                    This is the parent element !!!
                    <p>This is first paragraph!</p>
                    <span>first span box !</span>
                    <h2>heading 2!</h2>
                    <h3>heading 3!</h3>
                    <h4>heading 4!</h4>
                    <h5>heading 5!</h5>
                    <h6>heading 6!</h6>
                    <p>This is second paragraph!</p>
                </div>
            </body>
              
            </html>

    Resultado


    Método prev() & prevAll():

    El prev() es una función incorporada en jQuery que se utiliza para devolver el elemento hermano anterior del elemento seleccionado.

    Su Sintaxis es:

    $(selector).prev()

    Por otro lado, el prevAll() es un método incorporado en jQuery que se utiliza para devolver todos los elementos hermanos anteriores del elemento seleccionado.

    Su Sintaxis es:

    $(selector).prevAll()

    Ejemplo con prev():

    
            
            <html>
 
            <html>
 
            <head>
                <style>
                    .pre * {
                        display: block;
                        border: 2px solid lightgrey;
                        color: black;
                        padding: 5px;
                        margin: 15px;
                    }
                </style>
                <script src="https://ajax.googleapis.com/ajax/libs/
                             jquery/3.3.1/jquery.min.js"></script>
                <script>
                    $(document).ready(function() {
                        $("h3").prev().css({
                            "color": "black",
                            "border": "2px solid green"
                        });
                    });
                </script>
            </head>
             
            <body class="pre">
                <div>
                    This is parent element !
                    <p>This is first paragraph </p>
                    <span>First span box </span>
                    <h2>heading 2 !</h2>
                    <h3>heading 3 !</h3>
                    <p>This is the second paragraph and next sibling to h3 !</p>
                </div>
            </body>
             
            </html>

    Método prevUntil():

    El prevUntil() es un método incorporado en jQuery que se utiliza para encontrar todos los elementos hermanos anteriores entre dos elementos dados.

    Su Sintaxis es:

    $(selector1).nextUntil(selector2)

    Ejemplo:

    
            <html>
  
            <head>
                <style>
                    .sib * {
                        display: block;
                        border: 2px solid lightgrey;
                        color: black;
                        padding: 5px;
                        margin: 15px;
                    }
                </style>
                <script src="https://ajax.googleapis.com/ajax/libs/
                      jquery/3.3.1/jquery.min.js"></script>
                <script>
                    $(document).ready(function() {
                        $("li.start").prevUntil("li.stop").css({
                            "color": "black",
                            "border": "2px solid green"
                        });
                    });
                </script>
            </head>
              
            <body>
                <div style="width:400px;" class="sib">
                    <ul>
                        This is parent !!!
                        <li class="stop">list 1 !!!</li>
                        <li>first list !</li>
                        <li>second list !</li>
                        <li>third list !</li>
                        <li>fourth list !</li>
                        <li>fifth list !</li>
                        <li class="start">list 7 !!!</li>
                    </ul>
                </div>
            </body>
              
            </html>
