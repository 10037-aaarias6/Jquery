Descendants

jQuery es una biblioteca de JavaScript popular que simplifica la manipulación y navegación del árbol DOM (Modelo de Objetos del Documento) en páginas web. Una de las características principales de jQuery es su capacidad para realizar selecciones precisas de elementos HTML y manipular su contenido o estilo.

El travesía descendiente, también conocido como "descendant traversal" en inglés, se refiere a la capacidad de jQuery para seleccionar y manipular elementos descendientes de un elemento específico dentro del árbol DOM.

Existen dos métodos en jQuery que permiten realizar la travesía descendiente:


  • children()
  • find()

    • El método children()

    Devuelve todos los hijos directos del elemento seleccionado. Este método atraviesa un solo nivel hasta el árbol DOM.

    Ejemplo:

    
            <!DOCTYPE html>
            <html>
              
            <head>
                <style>
                    .descendants * {
                        display: block;
                        border: 2px solid lightgrey;
                        color: lightgrey;
                        padding: 5px;
                        margin: 15px;
                    }
                </style>
                <script src=
            "https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js">
              </script>
                <script>
                    $(document).ready(function() {
                        $("div").children().css({
                            "color": "green",
                            "border": "2px solid blue"
                        });
                    });
                </script>
            </head>
              
            <body>
              
                <div class="descendants" 
                    style="width:500px;">
                  current element
                    <p>child
                        <span>
                          grandchild
                      </span>
                    </p>
                </div>
            </body>
              
            </html>

    Resultado


    El método find()

    Devuelve elementos descendientes del elemento seleccionado y hasta el último descendiente.

    Ejemplo:

    
            <!DOCTYPE html>
            <html>
              
            <head>
                <style>
                    .descendants * {
                        display: block;
                        border: 2px solid lightgrey;
                        color: lightgrey;
                        padding: 5px;
                        margin: 15px;
                    }
                </style>
                <script src=
            "https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js">
              </script>
                <script>
                    $(document).ready(function() {
                        $("div").find("span").css({
                            "color": "green",
                            "border": "2px solid blue"
                        });
                    });
                </script>
            </head>
              
            <body>
              
                <div class="descendants" 
                    style="width:500px;">
                  current element
                    <p>child
                        <span>grandchild</span>
                    </p>
                </div>
              
            </body>
              
            </html>

    Resultado
